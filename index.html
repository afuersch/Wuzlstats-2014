What?

Wuzlstats is a simple web application to keep track of your offices Wuzl[1] score.

[1] Wuzln may also be called Tischfu√üball, Tisch-Kicken or something similar, depending on where you come from. But Wuzln is the correct way.

How?

You'll need IIS and .NET 4.5.1 to host this web application (built with ASP.NET MVC). Also, SQL Server is required for storing the scores.

The easiest way to get Wuzlstats up and running is to use the free tier of Azure:

  1. Fork the Wuzlstats repository on GitHub
  2. Log in to the Azure management portal. Create a free account if you don't have one yet.
  3. Create a new Website
    1. Use your fork on GitHub for automatic publishing
    2. Create a SQL Server database (the free 20 MB are more than enough). Use Db as the name for the connection string
  4. Done.
